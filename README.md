# angular-require-express
node(express) + angular + require + grunt，按需加载，自动压缩，前后分离，适合大型项目

首先需要安装node
当前版本：
node v6.7.0
npm 3.10.3

下载项目
git clone https://github.com/ooyutao/angular-require-express.git

进入项目
cd angular-require-express

安装依赖
npm i

启动项目
npm start

安装grunt服务
npm i -g grunt-cli

启动grunt监听代码变化
grunt
